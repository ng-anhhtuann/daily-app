
# Daily App

This app is built to track your activities and tasks in a time by yourself, which let u get a better habbit in checking to see and to add what u will do next :)

## Installation

Make sure you install `node` vs `react-native` and follow steps:
```bash
git clone https://github.com/ng-anhhtuann/daily-app.git
```
then open your IDE with the path to the project
```bash
npm install 
```
wait a bit and 
```bash
npm react-native run-ios 
```
enjoy here~~
## Addition
If you use Windows, use `choco` instead
